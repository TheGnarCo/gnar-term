# Keyboard Shortcuts

All shortcuts use `Cmd` on macOS and `Ctrl` on Linux unless noted otherwise.

## Workspaces

| Shortcut         | Action                 |
| :--------------- | :--------------------- |
| `Cmd+N`          | New workspace          |
| `Cmd+1`--`Cmd+8` | Jump to workspace 1--8 |
| `Cmd+9`          | Jump to last workspace |
| `Ctrl+Cmd+]`     | Next workspace         |
| `Ctrl+Cmd+[`     | Previous workspace     |
| `Shift+Cmd+W`    | Close workspace        |
| `Shift+Cmd+R`    | Rename workspace       |
| `Cmd+B`          | Toggle sidebar         |

## Surfaces (tabs)

| Shortcut         | Action       |
| :--------------- | :----------- |
| `Cmd+T`          | New tab      |
| `Shift+Cmd+]`    | Next tab     |
| `Shift+Cmd+[`    | Previous tab |
| `Ctrl+Tab`       | Next tab     |
| `Ctrl+Shift+Tab` | Previous tab |
| `Cmd+W`          | Close tab    |

## Split panes

| Shortcut          | Action                   |
| :---------------- | :----------------------- |
| `Cmd+D`           | Split right              |
| `Shift+Cmd+D`     | Split down               |
| `Alt+Cmd+Arrow`   | Focus pane directionally |
| `Shift+Cmd+Enter` | Toggle pane zoom         |
| `Shift+Cmd+H`     | Flash focused panel      |

## General

| Shortcut          | Action                      |
| :---------------- | :-------------------------- |
| `Cmd+P`           | Command palette             |
| `Cmd+K`           | Clear scrollback            |
| `Cmd+=` / `Cmd++` | Increase terminal font size |
| `Cmd+-`           | Decrease terminal font size |

Linux variants use `Ctrl+Shift+=` / `Ctrl+Shift+-` for font size (plain `Ctrl+=`/`Ctrl+-` would be intercepted by the terminal).
